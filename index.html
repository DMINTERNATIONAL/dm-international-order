🔥 실시간 연동 설정

모든 기기에서 주문을 실시간으로 공유하려면 Firebase 무료 계정이 필요해요. 5분이면 완료돼요!

1단계 — Firebase 프로젝트 만들기console.firebase.google.com 접속 → 프로젝트 추가 → 이름 입력(예: dm-orders) → 계속 → 프로젝트 만들기
2단계 — Realtime Database 만들기왼쪽 메뉴 → Realtime Database → 데이터베이스 만들기 → 테스트 모드로 시작 선택 → 사용 설정
3단계 — 데이터베이스 URL 복사데이터베이스 화면 상단에 https://프로젝트명-default-rtdb.firebaseio.com 형식의 URL을 아래에 붙여넣어 주세요.
