Firebase 무료 계정으로 모든 기기에서 실시간 주문 공유가 가능해요.

1단계 — Firebase 프로젝트 만들기console.firebase.google.com → 프로젝트 추가 → 이름 입력 → 만들기
2단계 — Realtime Database 만들기왼쪽 메뉴 → Realtime Database → 데이터베이스 만들기 → 테스트 모드 → 사용 설정
3단계 — URL 복사 후 아래 입력https://프로젝트명-default-rtdb.firebaseio.com
